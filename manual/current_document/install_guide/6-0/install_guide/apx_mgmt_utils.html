Installation Management Utilities

安装管理工具

用于安装和初始化inHybrid数据库系统的命令行管理工具的引用。

有关所有inHybrid数据库实用程序的完整参考，请参阅inHybrid数据库实用程序指南。

下面的inHybrid数据库管理工具位于$HDBHOME/bin中。
  • hdbactivatestandby

  • hdbaddmirrors

  • hdbcheckperf

  • hdbdeletesystem

  • hdbinitstandby

  • hdbinitsystem

  • hdbpkg

  • hdbscp

  • hdbssh

  • hdbssh-exkeys

  • hdbstart

  • hdbstop
