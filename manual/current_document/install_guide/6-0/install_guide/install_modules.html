Installing Additional Supplied Modules

安装附加提供的模块

inHybrid数据库发行版包括几个PostgreSQL和inHybrid来源的contrib模块，您可以选择安装这些模块。

每个模块通常打包为一个inHybrid数据库扩展。您必须在希望使用它的每个数据库中注册这些模块。例如，要在名为testdb的数据库中注册dblink模块，可以使用以下命令:

$ psql -d testdb -c 'CREATE EXTENSION dblink;'

若要从数据库中删除模块，请删除关联的扩展名。例如，要从testdb数据库中删除dblink模块:

$ psql -d testdb -c 'DROP EXTENSION dblink;'
注意:当您从数据库中删除模块扩展时，您在数据库中创建的任何引用模块中定义的函数的用户定义函数将不再工作。如果您创建了任何使用模块中定义的数据类型的数据库对象，inHybrid数据库将在您试图删除模块扩展时通知您这些依赖项。

您可以通过这种方式注册以下模块:

  • citext
  • dblink
  • fuzzystrmatch
  • gp_sparse_vector
  • hstore
  • pageinspect
  • pgcrypto
  • sslinfo

有关inHybrid数据库提供的模块的更多信息，请参考inHybrid数据库参考指南中的其他提供的模块。
