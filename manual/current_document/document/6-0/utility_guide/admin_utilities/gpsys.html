hdbsys1

hdbsys1

显示有关操作系统的信息。

概要

hdbsys1 [ -a | -m | -p ]
                
                hdbsys1 -? 
                
                hdbsys1 --version

描述

hdbsys1显示当前主机的平台和安装的内存（以字节为单位）。例如：

linux 1073741824

选项

-a（显示所有）
显示当前主机的平台和内存信息。这是默认选项。
-m（仅显示内存）
以字节为单位显示安装的系统内存。
-p（仅显示平台）
显示操作系统平台。可以是linux、darwin或sunos5。
-?（帮助）
显示在线帮助。
--version
显示此工具的版本。

示例

显示有关当前主机操作系统的信息：

hdbsys1

另见

hdbcheckperf
