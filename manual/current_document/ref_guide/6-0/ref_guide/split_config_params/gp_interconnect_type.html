gp_interconnect_type

设置用于inHybrid数据库interconnect流量的网络协议。 UDPIFC指定使用UDP和流量控制来实现interconnect流量，并且是唯一支持的值。

UDPIFC（默认值）指定使用UDP和流量控制来实现interconnect流量。 使用gp_interconnect_fc_method指定interconnect流控制方法。

使用TCP作为interconnect协议，inHybrid数据库的上限为1000个segment实例 - 如果查询工作负载涉及复杂的多切片查询时，上限小于该值。
