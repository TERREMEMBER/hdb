gp_instrument_shmem_size

为查询指标分配的共享内存量（以KB为单位）。 默认值为5120，最大值为131072。 启动时，如果gp_enable_query_metrics设置为on，则inHybrid数据库会在共享内存中分配空间以保存查询指标。 该内存被组织为标题和插槽列表。 所需的插槽数取决于并发查询的数量和每个查询的执行计划节点的数量。 默认值5120基于inHybrid数据库系统，该系统最多执行约250个并发查询，每个查询120个节点。 如果gp_enable_query_metrics配置参数关闭，或者插槽耗尽，则指标将保留在本地内存中而不是共享内存中。
