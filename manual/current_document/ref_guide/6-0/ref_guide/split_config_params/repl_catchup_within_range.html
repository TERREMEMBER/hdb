repl_catchup_within_range

对于inHybrid数据库主镜像，控制活跃从节点的更新。 如果walsender尚未处理的WAL段文件数超过此值，inHybrid数据库将更新活跃从节点。

如果段文件的数量不超过该值，则inHybrid数据库会阻止更新，以允许walsender处理文件。 如果已处理所有WAL段，则更新活动主站。 如果段文件的数量不超过该值，则inHybrid Database阻止更新到，以允许walsender处理文件。 如果已处理所有WAL段，则将更新活动的主控。

取值范围 默认值 设置分类
0 - 64 1 master
