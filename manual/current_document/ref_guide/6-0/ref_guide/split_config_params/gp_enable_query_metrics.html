gp_enable_query_metrics

启用查询指标的收集。 启用查询指标收集后，inHybrid数据库会在查询执行期间收集指标。 默认为关闭。

更改此配置参数后，必须重新启动inHybrid数据库才能使更改生效。

inHybrid数据库指标收集扩展在启用时，会将收集的指标通过UDP发送到inspur inHybrid Command Center代理1。

Note: 1指标收集扩展包含在inspur的inHybrid数据库的商业版本中。 inspur inHybrid Command Center仅支持inspur inHybrid数据库。
